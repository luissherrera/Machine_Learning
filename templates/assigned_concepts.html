📘 Assigned Classification Model – Basic Concepts

1. What the model is (Nearest Centroid)

Nearest Centroid is a simple supervised classification algorithm. Each class has a centroid, the average of all feature values in that class. A new data point is assigned to the class whose centroid is closest. This method is intuitive, fast, and works well when classes form well-defined groups.

2. How it works in general terms

First, the algorithm calculates the centroid for each class using the training data. Then, for any new observation, it measures the distance to each centroid and assigns the observation to the nearest class. Common distance metrics include Euclidean and Manhattan. It does not require iterative training, making it fast and easy to use.

3. Main characteristics

  • Simple and fast to train and predict.
  • Works best when classes are clearly separated and roughly spherical.
  • Non-parametric: no assumption about data distribution.
  • Handles multi-class problems naturally.
  • Low memory and computational cost, even for medium-sized datasets.

4. Advantages and limitations

  • Advantages: fast, easy to understand, few parameters, low memory usage, suitable for small/medium datasets.
  • Limitations: sensitive to outliers, performs poorly if classes overlap or have irregular shapes, less effective for high-dimensional sparse data without scaling.

5. Type of classification problems

Nearest Centroid is useful when classes are compact and well-separated. Works for binary and multi-class tasks, especially when features meaningfully contribute to separation. Typical applications: document classification, simple image recognition, preliminary medical diagnosis.

6. Real-world use example

Example: document classification using word frequency vectors. Another example: preliminary medical diagnosis based on measurements like blood tests or vital signs, allowing fast, interpretable classification.
