Student Dropout Prediction

Advanced analysis using Binary Classification to improve academic retention.

Problem

Many students leave their studies without warning, impacting both their personal future and institutional performance.

Machine Learning Type

Supervised Learning — Binary Classification

The model learns from historical student data to predict:
• 0 = No dropout
• 1 = Dropout risk

Algorithms

  • Random Forest: High accuracy and robust performance.
  • Logistic Regression: Simple, fast, and interpretable.
  • XGBoost: Efficient for handling complex datasets.

Input Data (Features)

  • Grades and GPA
  • Attendance records
  • Number of failed courses
  • Current semester / Study hours

Institutional Impact

Early Intervention

Identify at-risk students.

Better Support

Personalized guidance.

Retention

Improve performance.
