K-Means Clustering Concepts

Understand how unsupervised learning groups similar data points

1. What is Clustering?

Clustering is an unsupervised machine learning technique used to group a set of objects in such a way that objects in the same group (called a cluster) are more similar to each other than to those in other groups. It helps in discovering inherent structures or patterns in data without pre-existing labels.

2. What is K-Means?

K-Means is one of the most popular clustering algorithms. The "K" refers to the number of clusters you want to define. The algorithm iteratively assigns each data point to one of K groups based on the features that are provided. Data points are clustered based on feature similarity.

3. How does K-Means work?

  1. Initialization: Choose K initial centroids (cluster centers) randomly.
  2. Assignment: Assign each data point to the cluster whose centroid is closest.
  3. Update: Recalculate the centroids by computing the mean of all data points assigned to each cluster.
  4. Repeat: Repeat steps 2 and 3 until the centroids no longer change significantly or a maximum number of iterations is reached.
