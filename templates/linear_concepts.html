📘 4. Linear Regression – Basic Concepts

4.1 What Is Linear Regression?

Linear Regression is a statistical and machine learning method that models the linear relationship between a dependent variable (the value we want to predict) and one or more independent variables (the input features). It assumes that the relationship between inputs and output can be approximated by a straight line (in simple regression) or a hyperplane (in multiple regression).

The general form of a simple linear regression model is:

y = β₀ + β₁x + ε

Where:

  • y = the predicted (dependent) variable
  • x = the input (independent) variable
  • β₀ = the intercept (value of y when x = 0)
  • β₁ = the slope (rate of change of y per unit change in x)
  • ε = the error term (residual)

4.2 Independent and Dependent Variables

In the context of this project, the dependent variable is the IMDb score of a Netflix title. The independent variables are the input features used to predict this score. Examples include:

  • Release year of the content
  • Duration of the movie or series (in minutes)
  • Content rating (e.g., PG, R, TV-MA)
  • Number of votes received on IMDb

4.3 Training Data

Training data refers to the subset of the dataset used to fit the regression model. During training, the algorithm adjusts the slope (β₁) and intercept (β₀) to minimize the total prediction error across all training examples.

In this project, 80% of the Netflix dataset is used for training and 20% for testing.

4.4 Regression Line

The regression line is the best-fit straight line drawn through the data points that minimizes the sum of squared residuals (differences between actual and predicted values). This is achieved using the Ordinary Least Squares (OLS) method, which is the standard approach in linear regression.

4.5 Slope and Intercept

The slope (β₁) represents how much the predicted IMDb score changes for a one-unit increase in an independent variable, holding other variables constant.

The intercept (β₀) represents the baseline predicted score when all input features equal zero, though this value may not always have a practical interpretation depending on the feature scales.

4.6 Example of Application

Suppose we want to predict the IMDb score of a Netflix movie based solely on its release year. After training, the model might find that more recently released titles tend to have slightly higher ratings (e.g., β₁ = 0.03), meaning for each additional year of recency, the predicted score increases by 0.03 points.

If the intercept is β₀ = -50.5 and the release year is 2020, the model predicts:

IMDb Score = -50.5 + 0.03 × 2020 = 10.1 (capped at 10)

This simple example demonstrates the mechanics of the regression model.
