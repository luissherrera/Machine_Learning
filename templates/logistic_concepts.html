📘 Logistic Regression - Basic Concepts

1. What is Logistic Regression

Logistic Regression is a supervised machine learning algorithm used for classification tasks. Unlike linear regression, which predicts continuous numerical values, logistic regression predicts the probability that a given input belongs to a specific class.

The model uses a sigmoid function to transform its output into values between 0 and 1. These probabilities are then used to determine the final class label.

2. Difference between regression and classification

Regression models are used to predict continuous numerical values, such as prices, temperatures, or scores.

In contrast, classification models predict discrete categories or labels, such as "yes" or "no", "spam" or "not spam".

Although Logistic Regression contains the word "regression", it is mainly used for classification problems.

3. Binary classification

Binary classification refers to problems with only two possible outcomes.

  • 0 → Negative class
  • 1 → Positive class

This approach is widely used in applications such as spam detection, disease prediction, and fraud detection.

4. Target variable / class label

The target variable, also known as the class label, is the output that the model is trying to predict.

It represents the category assigned to each observation. For example, in a loan prediction model, the target variable could be approved or not approved.

5. Features / predictors

Features, also called predictors, are the input variables used by the model.

These variables help the model identify patterns and relationships. Examples include:

  • Age
  • Income
  • Work experience

6. Probability estimation

Logistic Regression produces probability values between 0 and 1.

For example, a prediction of 0.85 indicates an 85% probability that the observation belongs to class 1.

7. Decision boundary

The decision boundary is the line or surface that separates different classes.

It defines how the model distinguishes between categories based on the input data.

8. Classification threshold

The classification threshold is used to convert probabilities into class labels.

The most common threshold is 0.5:

  • If probability ≥ 0.5 → Class 1
  • If probability < 0.5 → Class 0

9. Real-world use example

Logistic Regression is widely used in the banking sector to predict whether a customer will repay a loan.

The model analyzes variables such as income, credit history, and debt level to estimate the probability of repayment and support decision-making.
