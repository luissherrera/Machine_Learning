Logistic Regression - Basic Concepts

  1. What is Logistic Regression?

    Logistic Regression is a supervised machine learning algorithm used for classifying data. Instead of predicting exact numbers, it gives the probability that something belongs to one class or another. For example, it can show the probability that an email is spam or that a client will pay a loan.

  2. Difference between Regression and Classification

    Regression models predict continuous values, like a person's salary or temperature. Classification models, like Logistic Regression, predict categories, such as pass/fail, buy/not buy, or sick/healthy.

  3. Binary Classification

    Binary classification works with two possible outcomes:

    • 0 → Negative class, meaning the event did not happen or the condition is not met.
    • 1 → Positive class, meaning the event happened or the condition is met.

  4. Target Variable / Class Label

    The target variable, or class label, is what we want to predict. For example, if we want to predict whether a client pays a loan, the target is “paid” or “not paid”.

  5. Features / Predictors

    Features or predictors are the inputs used to make predictions. Examples include age, income, payment history, or work experience. The model learns how these features relate to the target to make predictions.

  6. Probability Estimation

    Logistic Regression outputs probabilities between 0 and 1, showing how likely an instance belongs to a class. For example, a prediction of 0.8 means there is an 80% chance the instance belongs to the positive class.

  7. Decision Boundary

    The decision boundary is the line or surface that separates the classes in the feature space. Data on one side of the boundary is classified as one class, and data on the other side as the opposite class.

  8. Classification Threshold

    A threshold converts predicted probabilities into final class labels. Typically, 0.5 is used: probabilities ≥ 0.5 are assigned to class 1, and probabilities < 0.5 go to class 0.

    • If probability ≥ 0.5 → Class 1
    • If probability < 0.5 → Class 0

  9. Real-world Use Example

    Logistic Regression is used in finance, healthcare, and marketing. For example, a bank can predict if a client will pay a loan based on age, income, and debt. Hospitals can predict patient risk for certain diseases based on symptoms and medical tests.
